# Penetration Testing Career Roadmap

Penetration Testers (or Ethical Hackers) are authorized to simulate cyberattacks to identify and fix security vulnerabilities. This path focuses on the offensive mindset, technical exploitation, and professional reporting.

## üü¢ Level 1: The Foundations (Pre-Security)
Before you can break a system, you must understand how it functions.
* **Networking Basics:** TCP/IP stack, OSI Model, DNS, DHCP, and HTTP/S.
* **Operating Systems:** * **Linux:** Command line proficiency, file permissions, and bash scripting.
    * **Windows:** Active Directory basics, PowerShell, and registry.
* **Virtualization:** Setting up labs using VMware or VirtualBox.

## üü° Level 2: Security Essentials
Understanding the "Defense" to better execute the "Offense."
* **Web Security:** OWASP Top 10 (SQLi, XSS, CSRF, etc.).
* **Cryptography:** Hashing vs. Encryption, Symmetric vs. Asymmetric.
* **Security Principles:** CIA Triad, Principle of Least Privilege.

## üü† Level 3: Technical Skills & Tools
This is where the hands-on hacking begins.
* **Reconnaissance & Enumeration:** * *Tools:* Nmap, Recon-ng, Shodan.
* **Vulnerability Assessment:** * *Tools:* Nessus, OpenVAS.
* **Exploitation Frameworks:** * *Tools:* Metasploit, Armitage.
* **Web App Pentesting:** * *Tools:* Burp Suite (Crucial), OWASP ZAP, Gobuster.
* **Network Attacks:** Man-in-the-Middle (MITM), Password Cracking (Hashcat, John the Ripper).

## üî¥ Level 4: Advanced Specialization
* **Active Directory Hacking:** Kerberoasting, LLMNR poisoning, Lateral Movement.
* **Privilege Escalation:** Moving from a low-level user to Root/Administrator.
* **Wireless Pentesting:** Cracking WPA2/WPA3 (Aircrack-ng).
* **Scripting for Pentesters:** Python or Go for automating exploit payloads.

## üìù The Final Step: Reporting
A Penetration Tester is only as good as their report. You must be able to:
* Explain technical risks to non-technical stakeholders.
* Provide clear remediation steps for developers.
* Document the "Attack Chain" clearly.
